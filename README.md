## The gap

Tart frozen yogurt died with the fro-yo shop. Incumbents own sweetness + macros (Yasso, Halo Top), not tartness. MYKA proved demand at 240+ stores globally; Chara launched premium tart pints in LA at $12. Whitescape: nobody owns tart in packaged frozen, and campus is an untapped wedge.

We're the packaged, campus-native, cleaner-ingredient version.
